# plannig

This project include 3 part: 
# - scrapy
    For crawling a live events in different sites like tik8.com
    
# - shift_palnning
    This is composit of #Gnetic, #Simulated_Anealing, #Tabu_Search and #Guidline_Search algorithms together
    in #pandas #DataFrame to get best planning of nurses shifts by multi Constraints and Objectives

# - travel_palnning
    This is composit of #Gnetic, #Simulated_Anealing, #Tabu_Search and #Guidline_Search algorithms together
    in #numpy #matrices to get best planning of tour in a city for 1 or more days by multi Constraints and Objectives
